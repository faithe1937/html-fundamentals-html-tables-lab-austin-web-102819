This first row is almost always the only place where we use the th tag. The Table Data Tag: In the second row, we can start putting in our data. Here, we want to use the table data tag, instead of using the table header tag. The tag represents one table cell. So, for our party table, we can add in some names, phone numbers and items for guests to bring. An example row would look like the following: Daniel 845-282-4000 Guacamole As long as each bit of data in a is in the same position as the corresponding in the first row, it will show up in that column.
